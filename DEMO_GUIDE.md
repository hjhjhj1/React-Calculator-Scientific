# 演示指南

## 快速体验

### 1. 启动应用

应用已经在开发服务器上运行，访问地址：
- 通常是: `http://localhost:3000` 或 `http://localhost:3001`

### 2. 生成测试数据（推荐）

由于是新安装，没有历史记录。按照以下步骤生成演示数据：

1. 在计算器中输入任意表达式，例如 `2 + 3`
2. 点击 `=` 按钮执行计算
3. 计算器底部会出现两个新按钮：
   - 🔵 **查看分析**
   - 🔵 **生成测试数据**
4. 点击 **"生成测试数据"** 按钮
5. 会弹出提示："已生成50条测试数据！"

### 3. 查看数据分析

1. 点击 **"查看分析"** 按钮
2. 会打开一个大型模态窗口，显示分析面板

### 4. 探索分析面板

#### 标签页1：概览
- 查看4个关键指标卡片
- 查看常用函数表格
- 查看常用操作符表格
- 查看错误类型分布表格
- 查看使用习惯统计

#### 标签页2：图表
- 📊 **函数使用分布** - 饼图
- 📊 **操作符使用分布** - 饼图
- 📊 **错误类型分布** - 饼图
- 📊 **操作符使用频率** - 柱状图
- 📊 **近7日计算趋势** - 折线图（带填充）
- 📊 **24小时计算分布** - 折线图

#### 标签页3：历史记录
- 查看所有计算记录的表格
- 包含：序号、表达式、结果、时间、状态
- 失败的记录会用红色高亮显示

### 5. 使用时间段筛选

1. 在分析面板顶部，点击 **"显示筛选器"** 按钮
2. 选择开始日期和结束日期
3. 点击 **"应用筛选"**
4. 所有统计数据和图表会根据筛选条件更新
5. 点击 **"重置筛选"** 恢复全部数据

### 6. 导出报告

在分析面板顶部，有两个导出按钮：

#### 导出 JSON
- 点击 **"导出JSON"** 按钮
- 会下载一个包含完整统计数据和前50条记录的JSON文件
- 适合程序处理或详细分析

#### 导出 CSV
- 点击 **"导出CSV"** 按钮
- 会下载一个包含前100条记录的CSV文件
- 可以用Excel或WPS打开进行进一步分析

### 7. 进行实际计算

尝试使用计算器进行实际计算：

```
基本运算：
- 123 + 456
- 987 - 654
- 12 * 34
- 100 / 4
- 5 ^ 3（幂运算）

科学计算（先点击开关按钮展开科学面板）：
- 输入数字，然后点击 Sqrt（平方根）
- 输入数字，然后点击 Sin（正弦）
- 输入数字，然后点击 ln（自然对数）
- 输入数字，然后点击 e（指数）
- 输入数字，然后点击 X!（阶乘）
- 输入数字，然后点击 1/X（倒数）

组合运算：
- (10 + 5) * 3
- sqrt(16 + 9)
- 2 * sin(3.14 / 2)
```

每次计算后：
- 结果会显示在输入框
- 记录会自动保存到历史
- "查看分析"按钮会更新，可以随时查看最新统计

### 8. 测试错误处理

尝试输入错误的表达式，看看系统如何处理：

```
- 123 +（不完整的表达式）
- 10 / 0（除零错误）
- sqrt(-100)（负数平方根）
- abc + def（无效操作数）
```

系统会：
- 显示错误提示
- 记录错误类型
- 在分析面板中统计错误分布

## 技术亮点

1. **实时追踪** - 每次计算自动记录
2. **智能分析** - 自动识别函数和操作符
3. **可视化** - 多种图表类型，直观展示
4. **灵活性** - 支持自定义时间范围筛选
5. **可导出** - JSON和CSV两种格式
6. **本地化** - 完整的中文界面
7. **响应式** - 适配不同屏幕尺寸
8. **持久化** - 本地存储，数据不丢失

## 故障排除

### 如果看不到"查看分析"按钮
- 确保已经执行了至少一次计算
- 检查浏览器控制台是否有错误

### 如果图表不显示
- 确保Chart.js库正确安装（应该已经安装）
- 查看浏览器控制台的错误信息

### 如果导出失败
- 检查浏览器是否允许下载
- 检查是否有记录可以导出
